Indução Matemática

A prova por indução matemática é um método usado para provar que uma propriedade matemática é verdadeira para todos os valores de uma variável que atende a uma determinada condição. A prova por indução é composta por dois passos principais:

  1. Passo base: Prove que a propriedade é verdadeira para um valor inicial da variável.
  2. Passo de indução: Suponha que a propriedade é verdadeira para um valor $k$ da variável e mostre que a propriedade é verdadeira para o valor $k+1$.

Ou seja, a ideia é começar com um caso base e provar que ele é verdadeiro. Em seguida, supõe-se que a propriedade é verdadeira para um número arbitrário $k$ e tenta-se provar que ela também é verdadeira para $k+1$. Se esses dois passos são bem-sucedidos, então a propriedade é verdadeira para todos os valores maiores ou iguais ao caso base.

A prova por indução é frequentemente usada em problemas que envolvem números inteiros ou naturais, bem como em problemas que envolvem listas, sequências ou outras estruturas recursivas. É uma técnica poderosa para provar propriedades matemáticas que são válidas para uma infinidade de valores, mas que seriam difíceis de verificar individualmente para cada um desses valores.

Um exemplo simples de prova por indução é a seguinte afirmação: Para todo número natural $n$, a soma dos $n$ primeiros números inteiros é igual a $\dfrac {n(n+1)} 2$.

Para provar essa afirmação por indução, vamos seguir os dois passos mencionados anteriormente:

  1. Passo base: Verifique se a afirmação é verdadeira para $n = 1$.
  2. Passo de indução: Suponha que a afirmação é verdadeira para $n = k$ e prove que ela também é verdadeira para $n = k + 1$.

Suponha que a afirmação é verdadeira para $n = k$. Isto é, a soma dos primeiros $k$ números inteiros é igual a $\dfrac {k(k+1)} 2$. Agora, vamos considerar a soma dos primeiros $k + 1$ números inteiros:

$1 + 2 + 3 + ... + k + (k+1)$

Podemos reescrever essa soma como a soma dos primeiros $k$ números inteiros mais o próximo número inteiro:

$1 + 2 + 3 + ... + k + (k+1) = \dfrac {k(k+1)} 2 + (k+1) = \dfrac {(k+1)k} 2 + (k+1)\dfrac 2 2 = \dfrac {(k+1)(k+2)} 2 = \dfrac{(k+1)((k+1)+1)} 2$

E isso é exatamente o que queríamos provar. Portanto, a afirmação é verdadeira para $n = k + 1.$

Como o passo base e o passo de indução foram comprovados, podemos concluir que a afirmação é verdadeira para todos os valores naturais de $n$.
